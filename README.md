Projekt für die Social Network Analyse in der dazugehörigen Vorlesung an der DHBW (Jahrgang 2022)


Analyse des Wikispeedia Datensatzes, bereitgestellt durch die Stanford University: https://snap.stanford.edu/data/wikispeedia.html

Die Analyse orientiert sich stark an dem folgendem Artikel: https://studentwork.prattsi.org/infovis/labs/exploring-the-network-behind-wikispeedia/

Datenbereinigung findet nicht in R statt, sondern wurde zuvor mittels OpenRefine durchgeführt.
